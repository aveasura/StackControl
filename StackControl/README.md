# StackControl

StackControl is a lightweight stack size configuration mod for RimWorld.

## Features

- No HugsLib dependency.
- No Harmony dependency.
- Simple presets: Small x10, Medium x50, Large x100, Huge x500.
- Custom stack limits by item category.
- Safe handling for normal stackable items.
- Explicit support for body parts, implants and safe special items.
- Weapons, armor, apparel, corpses and quality-based items are left untouched.

## Presets

- Small: original stack size x10.
- Medium: original stack size x50.
- Large: original stack size x100.
- Huge: original stack size x500.
- Custom: exact maximum stack sizes by category.

## Multiplayer

StackControl should work in multiplayer if all players use the same mod version and the same settings.
Do not change StackControl settings during an active multiplayer session.

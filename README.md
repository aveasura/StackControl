# StackControl

A lightweight stack size configuration mod for RimWorld.

StackControl lets you increase item stack sizes using simple presets or exact custom limits by item category.

## Features

* No HugsLib required
* No Harmony required
* Simple presets: Small x10, Medium x50, Large x100, Huge x500
* Custom stack limits by category
* Supports resources, food, medicine, drugs, textiles, body parts, implants and selected safe special items
* Does not change weapons, armor, apparel, corpses or quality-based items
* Multiplayer-friendly when all players use the same settings

## Supported versions

* RimWorld 1.5
* RimWorld 1.6

## Notes

StackControl only increases stack sizes. Custom values will not reduce items below their vanilla stack limits.
